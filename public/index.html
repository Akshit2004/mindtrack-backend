This repository contains the API server (Express + Firebase Admin). Vercel only supports serverless/edge functions for backends and cannot host a long‑running Node.js server directly.

This static page just confirms the repo deployed to Vercel. The actual API should be deployed to a serverful host (for example Render, Railway, or Cloud Run), then configure your frontend to call that URL.

Local development: run npm run dev and hit http://localhost:5000/health.
